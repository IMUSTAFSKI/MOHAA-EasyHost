## Report a Bug

If something is broken or not working as expected, open a GitHub Issue.

**Go to:**

```
https://github.com/IMUSTAFSKI/MOHAA-EasyHost/issues
```

Click **New Issue** and choose **Bug Report**.

Please include:

- What you expected to happen
- What actually happened
- Your Windows version (Windows 10 or Windows 11)
- Your PowerShell version if you know it
- Any error message the script showed
- Steps to reproduce the problem

The more detail you give, the faster it gets fixed.

---

## Request a Feature

If you have an idea for something new, open a GitHub Issue.

**Go to:**

```
https://github.com/IMUSTAFSKI/MOHAA-EasyHost/issues
```

Click **New Issue** and choose **Feature Request**.

Please include:

- What you want the script to do
- Why it would be useful
- Any example of how it should work

Good feature requests are specific. Instead of:

```
Make hosting easier
```

Write something like:

```
Add a button that copies both the ZeroTier and Playit connect commands at once
```

## Contribute Code

Contributions are welcome.

If you want to fix a bug or add a feature, follow these steps:

---

### Step 1. Fork the repository

Click **Fork** at the top right of the GitHub page.

This creates your own copy of the project.

---

### Step 2. Create a branch

Name your branch after what you are doing:

```
fix-server-crash
add-zerotier-detection
update-readme
```

---

### Step 3. Make your changes

Keep changes focused. One fix or one feature per pull request.

If you are editing the PowerShell script:

- Keep the existing code style
- Test on a real MOHAA installation before submitting
- Do not include `MOHAA.exe`, `MOHAA_server.exe`, or any game files

---

### Step 4. Open a Pull Request

Go to your fork on GitHub and click **Compare & pull request**.

In the description, explain:

- What you changed
- Why you changed it
- How you tested it

---

### What Gets Accepted

Pull requests are more likely to be accepted if they:

- Fix a real bug with a clear description
- Add a feature that was already requested in Issues, usefull Feature or Tui enhancments 
- Follow the existing code style
- Do not add unnecessary dependencies
- Work on Windows 10 and Windows 11

## Guidelines

To keep this project clean and welcoming:

- Be respectful in all Issues, Discussions, and Pull Requests
- Write in English so everyone can follow the conversation
- Do not share or request pirated game files in this repository
- Do not open duplicate Issues — search first
- Do not spam feature requests — one clear request is better than ten vague ones
